Sample Census Custom Endpoints

You should have the following integration endpoints available on this server.

Name Description Endpoint URL
minimal A minimal endpoint that implements every method Census expects but doesn't push data. {{ site.environment.URL }}/.netlify/functions/minimal
postgres a fully functional integration that can write into PostgreSQL databases {{ site.environment.URL }}/.netlify/functions/postgres

Installation & Testing Instructions

These functions are currently deployed and waiting to be connected in Census

  1. Login to Census at app.getcensus.com
  2. Click on the Connections tab
  3. Click on Add Service and select Custom API
  4. Here you should enter the URL of the custom API you've created. You can paste the URLs of one of the integrations hosted on this server.
